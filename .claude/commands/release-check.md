# Release Check

Prepare a lightweight pre-release check for this project.

## Checklist
1. Camera open/capture flow works.
2. File upload flow works.
3. Result section renders all three status paths.
4. Disclaimer modal opens/closes correctly.
5. Console has no unhandled error.

## Required Output
- PASS/FAIL per item.
- If FAIL: root cause and smallest safe fix.
- Final go/no-go recommendation.
